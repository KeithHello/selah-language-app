服務商比較
針對 STT、AI 翻譯、TTS 三個環節，比較主流服務商的價格、品質和適用場景。
服務商價格延遲中文品質優點缺點
OpenAI Whisper推薦$0.006/min2-5s⭐⭐⭐⭐最便宜、中文好延遲較高
Deepgram$0.0043/min0.5-1s⭐⭐⭐最快（即時）、便宜中文不如 Whisper
Google Cloud STT$0.016/min1-3s⭐⭐⭐⭐⭐中文品質最好最貴
Azure Speech$0.017/min1-3s⭐⭐⭐⭐免費 5hr/月API 較複雜
建議：初期用 Whisper（最便宜）。需要即時轉寫時換 Deepgram。
服務商價格（/M tokens）翻譯品質優點缺點
GPT-4o-mini推薦入 $0.15 / 出 $0.60⭐⭐⭐⭐極便宜、速度快偶爾不夠道地
Claude Haiku入 $0.25 / 出 $1.25⭐⭐⭐⭐⭐翻譯最自然貴 2-3 倍

GPT-4o入 $2.50 / 出 $10.00⭐⭐⭐⭐⭐品質最高貴 20 倍
建議：用 GPT-4o-mini（每次 < $0.0001）。品質接近 GPT-4o 但成本低 20 倍。

Part 5
降本策略
四個經過設計的優化手段，可以把每用戶成本再降 50-70%。
🔁
音頻緩存 + 複用
同一句英文的 TTS 只需生成一次。相似中文可能產出相同翻譯，複用已有音頻。
節省 30-50% TTS
📦
批量預生成
冷啟動的預設句子庫提前批量生成翻譯+TTS，用戶加入時直接下載。
前 5 天零 API 成本
⏰
非高峰生成
用戶晚上錄音，TTS 可以深夜批量處理。第二天打開時音頻已準備好。
可用更便宜的批量方案
📊
用量分級
免費用戶每天 1 次錄音 + OpenAI TTS。付費用戶無限錄音 + ElevenLabs。
免費用戶 < $0.05/月

音頻播放 + 速度控制前端音頻已下載到本地$0

AI 翻譯 + 詞彙分析後端 AI需要 LLM 生成結構化輸出$0.0001/次
TTS 語音合成 ⚠️後端 AI需要高品質 AI 語音$0.001-0.015/次
句子排程後端需要全局安排學習時間線≈ $0
通知 + Widget前端iOS 本地排程$0
核心原則：能前端做的，絕不送後端。只有三件事必須後端做——聽（STT）、想（AI 翻譯）、說（TTS）。其他全部前端完成，確保離線可用、零延遲、零成本。
