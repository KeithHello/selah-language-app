Prototype 階段（現在）：使用 CSS + SVG 動畫，直接在 HTML 中實現。瀏覽器原生支持，互動性強，即時預覽。

iOS App 階段（未來）：使用 Lottie (JSON) 格式。優勢：
• 向量動畫，無限縮放不失真
• 文件極小（每個動畫 5-50KB）
• SwiftUI 原生支持（lottie-ios 庫）
• 可程式化控制（播放、暫停、跳轉、速度、循環）
• 設計師用 After Effects 製作，開發者直接導入

替代方案：Rive（互動狀態機，適合複雜角色）、SpriteKit（Apple 遊戲框架，適合複雜動畫）
// SwiftUI + Lottie 偽代碼 import Lottie struct SeedPetView: View { @State private var currentAnimation: String = "seed-idle-float" var body: some View { LottieView(animation: .named(currentAnimation)) .playing(loopMode: .loop) .frame(width: 80, height: 80) } // 觸發動畫 func triggerAnimation(_ event: PetEvent) { switch event { case .listenStart: currentAnimation = "seed-listen-tilt" case .quizCorrect: currentAnimation = "seed-bounce-happy" case .idle: currentAnimation = "seed-idle-float" } } }
