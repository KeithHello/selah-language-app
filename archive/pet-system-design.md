# Selah 寵物系統設計

> 小鳥精靈的四維度行為系統與五階段視覺演進
> 本文檔為技術設計文件，具體數值不對用戶公開

## 1. 核心概念

Selah 的寵物是一隻小鳥精靈，隨用戶的語言學習旅程一起成長。四個維度（飽食/心情/健康/親密）各自影響寵物的不同視覺層面，組合出豐富的表現。寵物的外形隨 XP 累積經歷五個進化階段。

---

## 2. 寵物進化階段

| 階段 | XP 閾值 | 大約時間 | 小鳥外觀 |
|------|---------|---------|---------|
| 蛋 | 0 | Day 0 | 淡奶油色橢圓蛋體，微微搖晃，偶爾出現裂痕紋 |
| 幼體 | 50 XP | Day 1-2 | 圓潤小鳥，天藍色，大眼睛，小呆毛，翅膀很小不會拍 |
| 成長期 | 300 XP | ~W1-2 | 體型增大，呆毛變長，翅膀能拍動，尾羽開始生長 |
| 成熟期 | 1000 XP | ~W3-4 | 完整體型，華麗尾羽，羽毛色彩更深，能短暫飛行 |
| 傳說型 | 3000 XP | ~W6+ | 金色羽尖，身周環繞小星星/光粒子，最華麗形態 |

### 各階段外觀細節

**蛋形態**
- 形狀：橢圓，上窄下寬
- 顏色：#FFE8D6 → #FFD4BC 漸變
- 動畫：緩慢左右搖晃，偶爾出現一道裂痕紋然後消失
- 高光：左上角有柔和反光

**幼體形態**
- 體型：非常圓潤，頭身比 1:1
- 顏色：#B8D9F0（淡天藍）
- 特徵：大大的圓眼睛、短小的呆毛（1-2 根）、很小的翅膀貼在身體兩側
- 動畫：輕微 floating，偶爾眨眼
- 表情：呆萌，嘴巴是小的三角形鳥嘴

**成長期形態**
- 體型：比幼體大 20%，頭身比 1:1.3
- 顏色：#8FC4E8（更鮮明的天藍）
- 特徵：呆毛變 3 根且更長、翅膀明顯增大能拍動、尾羽開始生長（2-3 根短羽）
- 動畫：floating 幅度更大，翅膀有節奏拍動
- 新增：腮紅更明顯，偶爾會歪頭

**成熟期形態**
- 體型：比幼體大 40%，頭身比 1:1.5
- 顏色：#6AADDB（深天藍）+ 胸腹 #E8F4FC（淡白藍）
- 特徵：華麗的長尾羽（4-5 根）、翅膀完全展開、頭頂冠羽
- 動畫：可以短暫飛行（在畫面中飛一圈再回來）
- 新增：羽毛有光澤感，眼睛更有神采

**傳說型形態**
- 體型：與成熟期相同，但視覺效果華麗
- 顏色：#6AADDB 基礎上羽尖泛金色 #F0D080
- 特徵：身周環繞小星星/光粒子、羽毛邊緣有彩虹光澤
- 動畫：持續的輕微發光脈動，飛行時拖尾光效
- 新增：偶爾頭頂冒出用戶最近學的句子泡泡

## 3. 四維度系統

### 3.1 飽食度 Hunger

**定義**：精靈的長期養分來源，代表用戶的持續學習投入。

| 指標 | 數值 |
|------|------|
| 自然衰減 | 每 24h -15 |
| 盲聽完整輪 | +8 |
| 夜間預習 | +10 |
| 跟讀一輪 | +5 |
| 語料產出 | +6 |

**影響：動作活力**

| 數值區間 | 表現 |
|----------|------|
| 80-100 | 活潑彈跳，翅膀高頻拍動，偶爾在畫面中飛來飛去 |
| 50-79 | 正常 floating，偶爾拍翅 |
| 20-49 | 動作變慢，floating 幅度縮小，翅膀下垂 |
| 0-19 | 幾乎不動，趴在地上，偶爾抬頭看你 |

### 3.2 心情 Mood

**定義**：精靈的即時情緒，反映用戶當下的學習體驗。

| 指標 | 數值 |
|------|------|
| 自然衰減 | 每 12h -10 |
| 檢測通過 | +15 |
| 晚安語 | +10 |
| 靈光乍現 | +20 |
| 預測成功 | +12 |
| 檢測卡住 | 0（不扣，面對摩擦本身就值得鼓勵） |

**影響：表情**

| 數值區間 | 表現 |
|----------|------|
| 80-100 | 瞇瞇眼 + 嘴巴上揚 + 腮紅加深 + 頭上冒愛心/音符 |
| 50-79 | 正常圓眼 + 微笑 |
| 20-49 | 圓眼 + 平嘴，表情中性 |
| 0-19 | 下垂眼 + 嘴角下彎 + 頭上冒小烏雲 |

### 3.3 健康 Health

**定義**：精靈狀態的最後防線。平時不受影響，只有當用戶長期忽視精靈時才會下降。

| 指標 | 數值 |
|------|------|
| 自然衰減 | 平時不扣 |
| 觸發條件 | 飽食 AND 心情同時 < 20 持續 24h → -10/天 |
| 恢復方式 | 完成任何學習行為 → +20 |

**影響：外觀狀態**

| 數值區間 | 表現 |
|----------|------|
| 50-100 | 正常，毛色鮮亮 |
| 20-49 | 毛色變灰淡，身上出現小繃帶貼圖 |
| 0-19 | 沉睡態 — 閉眼、身體半透明、浮動停止，畫面加一層薄霧。緊急信號。 |

### 3.4 親密度 Bond

**定義**：用戶與精靈之間的長期羈絆。隱藏維度，用戶只知道「越來越親近」，不知道具體計算方式。

| 行為 | Bond 變化 |
|------|----------|
| 每日任務全完成 | +2 |
| Streak 每 +1 天 | +1 |
| 錄晚安語 | +3 |
| 靈光乍現 | +5 |
| 盲聽完整輪（3遍+跟讀） | +2 |
| 自由記錄語料 | +2 |
| 連續 3 天沒互動 | -3/天 |
| 沉睡喚醒後 | 重置到 max(Bond×0.5, 20) |

**影響：互動行為（解鎖制）**

| 數值區間 | 解鎖表現 |
|----------|---------|
| 0-19（陌生） | 精靈面朝前方，不會主動看你 |
| 20-39（熟悉） | 偶爾轉頭看你，點一下會歪頭 |
| 40-59（信賴） | 會在畫面上飛到你附近，點一下會開心轉圈 |
| 60-79（親密） | 名字旁邊冒小對話泡泡（隨機顯示今天學的英文），點一下會唱歌 |
| 80-100（靈魂） | 外觀出現小光暈 + 小星星環繞，點一下會模仿你最近錄的晚安語 |

## 4. 四維度組合效果

四個維度同時作用，產生豐富的組合表情：

- **高飽食 + 低心情**：動作活潑但表情悶悶不樂 → 用戶會好奇「怎麼了？」
- **低飽食 + 高心情**：開心但動作無力 → 像是餓著肚子還在笑，讓人心疼
- **高親密 + 低健康**：想靠近你但力氣不夠 → 飛到身邊但趴著不動
- **全維度低**：沉睡態，緊急信號

## 5. 階段進化對表現的影響

同一套四維度行為系統，隨寵物階段進化而表現更豐富：

| 階段 | 動作活力表現 | 表情表現 | 互動行為 |
|------|-------------|---------|---------|
| 蛋 | 搖晃幅度 | 無表情 | 點擊搖晃更劇烈 |
| 幼體 | 基本 floating + 跳 | 眨眼 + 嘴型 | 點擊歪頭 |
| 成長 | 拍翅 + 短距離飛 | 完整表情系統 | 歪頭 + 轉圈 |
| 成熟 | 完整飛行路線 | 表情 + 頭頂裝飾 | 飛行 + 對話泡泡 |
| 傳說 | 華麗飛行 + 光效 | 表情 + 光效 | 模仿語音 + 主動對話 |

## 6. 用戶可見描述（幫助面板文案）

以下為幫助面板中顯示的文案。故意模糊化具體數值，鼓勵用戶自行探索：

**🍎 飽食度**
> 精靈的養分。一起聆聽、預習、跟讀，都能餵飽牠。忘記餵的話，精靈會越來越沒精神喔。

**💗 心情**
> 精靈的情緒。練習順利、錄晚安語都會讓牠開心。就算卡住了也不用擔心——面對挑戰本身就值得鼓勵。

**🌿 健康**
> 精靈的最後防線。平時不用操心，只有其他兩項都低落太久才會受影響。看到精靈變得灰灰的，就是提醒你了。

**🔮 親密度**
> 你和精靈之間的羈絆。一起經歷得越多，連結越深。具體怎麼培養？試試看就知道了 ✨

**⭐ 成長值 XP**
> 推動精靈進化的力量。每一次練習、每一次嘗試，都在累積。具體獲得方式？在學習中慢慢發現吧。

## 8. 沉睡與喚醒機制

當健康值降到 15 以下，精靈進入沉睡態：

- **視覺**：閉眼、身體半透明、浮動停止、畫面薄霧
- **通知**：Push notification「小豆 在等你回來...」
- **喚醒**：完成任何一個學習行為即可喚醒
- **喚醒後**：Bond 重置到 max(Bond×0.5, 20)，其他維度恢復到 30
- **設計意圖**：不是懲罰，是溫柔的提醒。「精靈想你了」比「你被扣分了」更有動力
