Stage 1 — 種子
Day 1-3 · 圓潤種子 + 眼睛 · 無芽無葉
最簡單的階段。種子是一顆暖琥珀色的圓球，只有兩個小點眼睛和微笑。沒有葉子、沒有芽——所有動畫只能靠身體的搖晃、滾動、擠壓和眼睛表情來表達。這正是它的魅力：用最小的視覺語言傳達最多的情感。

⚙️ 開發實現指南
動畫素材庫的使用方式。所有動畫在 App 中通過 AnimationManager 統一管理，根據當前 Stage + 觸發事件 + 隨機因子選擇播放。
// 偽代碼：動畫觸發邏輯 class SeedAnimationManager { currentStage: 1-4 // 根據 Day 計算 animationPool: Map<Stage, Animation[]> // 每次 App 打開或學習事件時調用 triggerAnimation(event: EventType) { const stage = this.currentStage const candidates = this.animationPool[stage] .filter(a => a.trigger === event.type) .filter(a => a.priority <= currentPriorityThreshold) // P0 動畫直接播放，P1/P2 隨機選一個 const anim = this.selectByPriority(candidates) // 檢查是否與當前動畫衝突 if (!anim.isLoop || !this.isPlaying) { this.play(anim) } } // Idle 動畫由計時器觸發（每 3-8 秒隨機選一個） scheduleIdleAnimations() { setInterval(() => { if (noUserAction for 3s) { this.triggerAnimation({ type: 'idle' }) } }, random(3000, 8000)) } } // CSS 動畫命名規則 // .seed-s{stage}-{category}-{name} // 例：.seed-s2-action-listen-leaf-up // 例：.seed-s4-idle-bloom-sway // 例：.seed-s3-time-night-blanket
📊 優先級說明
P0 — 必須實現
核心體驗動畫。沒有這些，種子會感覺「死的」。MVP 階段必須全部實現。
P1 — 應該實現
豐富度動畫。有了這些，種子才有「個性」。v1.1 前實現。
P2 — 可以延後
驚喜動畫。讓用戶感到「哇！」。可以逐步加入。
